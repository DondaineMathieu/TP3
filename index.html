SQL - Structured Query Language

Le SQL (Structured Query Language) est un langage permettant de communiquer avec une base de données. Ce langage informatique est notamment très utilisé par les développeurs web pour communiquer avec les données d’un site web. Par exemple, pour stocker les informations des utilisateurs (identifiants, mot de passe etc...)
Il est aujourd'hui reconnu par la plupart des SGBD (Système de Gestion de Base de Données)
Ce travail est souvent combiné avec d'autres langages tels que le PHP et l'HTML.
On retrouve beaucoup de tutoriels sur internet pour utiliser et compléter une base de données, par exemple : sql.sh, W3Schools, Developpez.com. Sachant que ce langage est composé de commandes "simple" il est très important d'avoir un bon sens de la structure et de l'organisation pour gérer au mieux la base de données

Source : sql.sh & Wikipedia
